Pasta reservada para investigacoes pontuais do projeto.

Use esta pasta para scripts temporarios que ajudem a responder perguntas como:

- quais valores existem em um filtro especifico
- quais nomes sao parecidos, mas nao batem exatamente
- quais registros ficaram de fora em um cruzamento
- quais casos precisam revisao manual

Ideia:

- inspecoes = validacoes mais estaveis do projeto
- investigacoes = consultas exploratorias e pontuais
